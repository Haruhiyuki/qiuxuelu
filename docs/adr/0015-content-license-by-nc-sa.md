# ADR-0015：内容协议由 CC BY-SA 4.0 改为 CC BY-NC-SA 4.0

状态：已接受
日期：2026-06-16
关联：ADR 无前置；涉及 PRD §7（注册同意）、数据导出（worker/db export）

## 背景

站点内容默认许可原为 **CC BY-SA 4.0**。出于「内容可自由共享、但不被商用搬运」的考量，改为
**CC BY-NC-SA 4.0**（在署名 + 相同方式共享之外，增加「非商业」限制）。

## 决策

- 内容默认许可改为 **CC BY-NC-SA 4.0**：署名（BY）+ 非商业（NC）+ 相同方式共享（SA）。
- 全站对外文案、链接（`creativecommons.org/licenses/by-nc-sa/4.0/`）、导出元数据
  （`CONTENT_LICENSE='CC-BY-NC-SA-4.0'`）统一更新。
- **同意版本号上调**：`LICENSE_CONSENT_VERSION` 由 `CC-BY-SA-4.0/v1` 改为
  `CC-BY-NC-SA-4.0/v1`。据 consentGate 语义，**存量用户的同意版本不再匹配，会在下次受保护操作
  （发布/评论/协作）前被引导重新确认新许可**——这是许可变更应有的留痕与再授权。

## 取舍与后果

- NC 与 BY-SA 的 ShareAlike 不互通：本许可为**自此向后**的默认许可与再授权口径；既有内容的
  历史署名与修订凭证不变。早期测试阶段、用户基数小，重新确认成本可忽略。
- 维基百科用 BY-SA，本站不再与其「同协议」——文案中相应的「同维基百科」表述已移除。
- 升级阈值、双线权限等其余机制不受影响。
